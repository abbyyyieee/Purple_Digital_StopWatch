# Purple_Digital_StopWatch
A simple web-based stopwatch built with HTML, CSS, and JavaScript.   It shows hours, minutes, seconds, and milliseconds, and lets you start, pause, and reset the timer.

## Features

- Displays time in `HH:MM:SS:MS` format  
- Start, pause, and reset controls  
- Stylish design with CSS  

## How to Download and Use

1. Click the **Code** button on this GitHub repository.  
2. Select **Download ZIP** and extract the files.  
3. Open the extracted folder.  
4. Double-click `index.html` or open it in a web browser.  
5. Use the on-screen buttons to control the stopwatch.
